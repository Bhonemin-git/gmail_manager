VIDEO BACKGROUND SETUP
======================

Place your galaxy_background.mp4 file in this directory for the login page background.

Requirements:
- File name: galaxy_background.mp4
- Format: MP4 (H.264 codec recommended)
- Recommended resolution: 1920x1080 or higher
- Optimized for web playback (compressed for faster loading)

The video will:
- Autoplay on page load
- Loop continuously
- Play muted (for autoplay compatibility)
- Cover the full background area
- Display behind the login card with a semi-transparent overlay
